### PCB Fault Detection with YOLO

This project uses YOLO object detection to identify possible faults on printed circuit boards (PCBs) and integrated circuits (ICs). The goal is to train and run a computer vision model that can detect defects such as missing components, misaligned components, solder bridges, and damaged traces.

The project is designed to run on an NVIDIA Jetson platform for edge AI inference.
